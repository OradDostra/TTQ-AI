# TTQ-AI

Centralized repository for managing AI projects, skills, tools, and system prompts across the company.

## Structure

```
TTQ-AI/
├── projects/          # Individual AI projects
├── shared/            # Shared resources across all projects
└── templates/         # Starter templates for new projects
```

## Projects

| Project | Description |
|---------|-------------|
| [AI-Analyst](./projects/AI-Analyst/) | AI-powered data analysis agent |

## Getting Started

1. Browse an existing project under `projects/`
2. Use templates from `templates/` to create new skills, tools, or system prompts
3. Place shared resources in `shared/` for cross-project reuse
