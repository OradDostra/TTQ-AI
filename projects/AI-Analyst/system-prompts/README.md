# System Prompts

Versioned system prompts for the AI-Analyst across different roles and environments.

## Naming Convention

`<role-or-env>.md` — e.g., `analyst-base.md`, `analyst-finance.md`, `analyst-prod.md`

## Guidelines

- Keep prompts modular — compose from shared base prompts where possible
- Track every change via git commits with a clear message describing what changed and why
- See [`/shared/system-prompts/`](/shared/system-prompts/) for base prompts to extend
